# Caderno Temático: Inteligência Artificial Generativa aplicada aos estudos

## 1. Contexto e Objetivos

Este projeto foi desenvolvido como parte de um desafio prático da DIO, com o objetivo de criar um Caderno Temático utilizando o NotebookLM como ferramenta de apoio à aprendizagem ativa.

O tema escolhido foi **Inteligência Artificial Generativa aplicada aos estudos**, por ser um assunto atual e diretamente relacionado ao uso de IA como apoio para organização, revisão e aprofundamento de conteúdos.

### Objetivos de estudo

- Compreender o conceito de Inteligência Artificial Generativa.
- Explorar como ferramentas de IA podem apoiar a aprendizagem ativa.
- Praticar curadoria de fontes confiáveis.
- Criar prompts estratégicos para extrair melhores respostas.
- Consolidar o aprendizado em um miniguia de estudo.

## 5. Miniguia de Estudo

### 5.1 Resumo Estruturado

A Inteligência Artificial Generativa é uma área da IA voltada para a criação de novos conteúdos, como textos, imagens, códigos, resumos e respostas personalizadas.

No contexto educacional, pode ser usada para apoiar a organização de ideias, criação de perguntas de revisão, explicação de conceitos complexos e comparação entre diferentes fontes.

Entretanto, seu uso exige pensamento crítico. A IA pode cometer erros, generalizar respostas ou deixar de considerar detalhes importantes. Por isso, é essencial validar as respostas com fontes confiáveis.

### 5.2 Glossário

| Termo | Definição |
|------|-----------|
| Inteligência Artificial | Área da computação que busca criar sistemas capazes de realizar tarefas associadas à inteligência humana. |
| IA Generativa | Tipo de IA capaz de gerar novos conteúdos a partir de dados e padrões aprendidos. |
| Prompt | Instrução enviada para uma ferramenta de IA. |
| Engenharia de Prompt | Processo de testar e melhorar comandos para obter respostas mais úteis. |
| Curadoria de Fontes | Seleção criteriosa de materiais confiáveis para estudo. |
| NotebookLM | Ferramenta de IA do Google voltada para estudo e análise de fontes. |
| Aprendizagem Ativa | Estratégia em que o estudante participa de forma crítica e prática do próprio aprendizado. |
